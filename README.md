# نظام إدارة الموظفين (HR App) - نسخة الاستضافة

تطبيق ويب بسيط وسريع لإدارة الموظفين والطلبات، مصمم للعمل على الاستضافات المشتركة (مثل Hostinger).

## المميزات
- واجهة عربية بالكامل (RTL).
- تصميم متجاوب باستخدام Bootstrap 5.
- نظام صلاحيات (Admin, Manager, Employee).
- إدارة الموظفين (إضافة، مراجعة، اعتماد).
- نظام طلبات الإجازات.
- أمان عالي باستخدام PDO Prepared Statements وتشفير كلمات المرور.

## متطلبات التشغيل
- PHP 7.4 أو أحدث.
- MySQL أو MariaDB.
- تفعيل موديل PDO في PHP.

## طريقة التثبيت على Hostinger
1. قم بإنشاء قاعدة بيانات MySQL جديدة من لوحة تحكم Hostinger (hPanel).
2. قم باستيراد الملف `sql/database.sql` داخل قاعدة البيانات التي أنشأتها (باستخدام phpMyAdmin).
3. ارفع ملفات المشروع بالكامل إلى مجلد `public_html` أو مجلد فرعي داخله.
4. قم بتعديل بيانات الاتصال في ملف `includes/config.php`:
   - `DB_NAME`: اسم قاعدة البيانات.
   - `DB_USER`: اسم مستخدم قاعدة البيانات.
   - `DB_PASS`: كلمة مرور قاعدة البيانات.
   - `BASE_URL`: المسار الذي رفعت فيه المشروع (مثلاً `/` إذا كان في المجلد الرئيسي).

## بيانات الدخول الافتراضية
- **اسم المستخدم:** `admin`
- **كلمة المرور:** `admin123`

*ملاحظة: يرجى تغيير كلمة المرور فور تسجيل الدخول لأول مرة.*

## بنية المشروع
- `assets/`: ملفات التنسيق والبرمجة النصية (CSS/JS).
- `auth/`: صفحات تسجيل الدخول والخروج.
- `employees/`: صفحات إدارة واعتماد الموظفين.
- `includes/`: ملفات الإعدادات والاتصال والقوالب المشتركة.
- `leaves/`: صفحات طلبات وإدارة الإجازات.
- `sql/`: ملف قاعدة البيانات.
